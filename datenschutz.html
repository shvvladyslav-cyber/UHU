Datenschutzerklärung

Diese Datenschutzerklärung informiert über die Verarbeitung personenbezogener Daten auf der Plattform „UHU Kassel“.

1. Verantwortlicher

UHU Kassel
Website: https://www.uhu.digital
Kontakt: Telegram @UHU_help

2. Verarbeitete Daten

Bei Nutzung der Plattform können folgende Daten verarbeitet werden:

  • Ausgewählte Dienstleistung
  • Stadtteil / Einsatzort
  • Gewünschter Zeitpunkt
  • Budgetangaben
  • Freitext-Beschreibung der Aufgabe

3. Zweck der Verarbeitung

Die Daten werden ausschließlich zur Vermittlung und Organisation von Dienstleistungen verwendet.

4. Weitergabe an Dritte

Die Daten werden an den jeweiligen Ausführenden weitergeleitet, um den Auftrag durchführen zu können. Eine darüberhinausgehende Weitergabe erfolgt nicht.

5. Nutzung von Google Apps Script

Zur technischen Verarbeitung der Anfragen wird Google Apps Script (Google LLC) eingesetzt. Dabei werden Daten auf Servern von Google verarbeitet.

6. Nutzung von Telegram

Bei Kontaktaufnahme über Telegram gelten zusätzlich die Datenschutzbestimmungen von Telegram Messenger Inc.

7. Speicherdauer

Die Daten werden nur so lange gespeichert, wie es für die Bearbeitung der Anfrage erforderlich ist.

8. Rechte der Nutzer

Nutzer haben das Recht auf Auskunft, Berichtigung und Löschung ihrer personenbezogenen Daten.

9. Kontakt

Datenschutzanfragen können jederzeit über Telegram @UHU_help gestellt werden.
